# Java Task Manager

A simple command-line app to add, list, update, and delete tasks.

## Usage

1. Compile:  
   javac src/*.java

2. Run:  
   java -cp src Main

## Features

- Add, list, update, and delete tasks by ID
- Each task has a description, status, and timestamps

## Requirements

- Java 8+
